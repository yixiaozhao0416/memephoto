★ 开始之前 | Before You Start
确认你的环境已准备好
🌐
现代浏览器
推荐 Chrome 90+、Edge 90+、Firefox 88+、Safari 14+。不支持 IE。
📷
摄像头设备
笔记本内置摄像头、USB 外接摄像头均可。手机前置摄像头也支持。
🔆
良好光线环境
面朝窗户或台灯。避免逆光和顶光。光线是影响照片质量最关键的因素。
🔒
摄像头权限
首次使用时浏览器会弹出授权请求，点"允许"即可。照片不会上传到服务器。
💡
隐私保障：MemePhoto 是纯前端应用，所有图像处理均在你的浏览器本地完成。点击"启动摄像头"后，摄像头画面只在你的屏幕上显示，不会传输到任何服务器。

1🔢
选择拍照格数（2 / 4 / 6 / 9）

页面左侧"SELECT MODE"区域有四个按钮，分别对应不同的拍照格数。点击后右侧预览区会出现对应数量的空格占位。

📐
格数选择建议：
· 2格 — 一张正经一张搞笑，对比感强
· 4格 — 最经典的大头贴条，竖版打印感
· 6格 — 适合和朋友一起拍，每人有更多格
· 9格 — 最多表情变化，做表情包首选
2🎨
选择边框颜色

点击色块选择边框颜色。选中的色块会出现橙色高亮边框。颜色会实时更新到右侧预览区的相框边框上。

选择边框颜色
↑ 白色已选中（橙色高亮）
3📏
选择边框厚度

下拉菜单有两个选项：细边框（简约感）和厚边框（日系大头贴感）。厚边框在下载后的图片中效果更明显。

💡
厚边框 + 白色 是最接近真实日本 Purikura 大头贴机效果的组合，推荐第一次尝试。

中间面板顶部"SELECT STICKER"区域显示两个快速贴纸缩略图。点击任意一个即可选中，摄像头画面上方会实时叠加预览效果。

↑ 第2个已选中（蓝色边框）；点"..."打开更多贴纸
2✨
打开更多贴纸库

点击快速贴纸旁边的 "..." 按钮，弹出完整贴纸库弹窗，里面包含 15+ 种贴纸，支持翻页浏览。

贴纸库还有两个特殊选项："空白贴纸"（清除当前贴纸）和 "上传贴纸"（上传你自己的 PNG 图片作为贴纸）。

🎨
自定义贴纸上传要求：PNG 格式，透明背景效果最好。上传后贴纸会覆盖整个拍摄区域，建议图片比例接近 5:4（横向）。
3🎛️
调节亮度与对比度

摄像头画面下方有两个拖动滑块：

亮度（Brightness）：范围 0.7–1.3。默认值 1.0。拉到 1.1–1.2 可以让肤色更通透，超过 1.3 会过曝。

对比度（Contrast）：范围 0.7–1.3。默认值 1.0。轻微提升到 1.1 让五官更立体，不建议超过 1.2。

调整后实时预览，拍摄时这些参数会被应用到照片上。

★ STEP 3 — 启动摄像头与拍照
中间面板底部：操作按钮区
1📡
点击"启动摄像头"

点击中间面板底部的"启动摄像头"按钮。浏览器会弹出权限询问弹窗，选择"允许"。

权限允许后，摄像头画面会出现在中间的像素电脑框架内。画面是镜像显示的（和对着镜子一样），这是正常现象——拍出来的照片也是镜像，视觉上最自然。

🔒
如果权限弹窗没有出现：点击浏览器地址栏左边的锁形图标 🔒，找到"摄像头"权限，手动设置为"允许"，然后刷新页面。
😊 ▶ 摄像头已启动
2⏱️
点击"拍照"— 3秒倒计时

点击"拍照"按钮后，摄像头画面上会出现 3 → 2 → 1 → 0 的倒计时数字，同时屏幕会短暂闪白光（模拟闪光灯效果），然后自动拍下当前画面。

每次点击"拍照"只拍一格。需要拍几格就点几次——比如选了4格模式，需要点4次"拍照"按钮，每次可以换不同的表情或姿势。

3
⏱️
3秒用来做什么：深吸一口气 → 摆好姿势 → 准备表情 → 闪光。利用这3秒预热，比"咔嚓"即时拍的表情自然很多。
3🔁
重复拍摄，直到拍满所有格子

拍完一格后，照片会自动出现在右侧预览区的对应格子中。继续点击"拍照"拍下一格，每格可以换不同的表情、姿势，或者移动位置。

当所有格子都拍满后，再点击"拍照"会提示"已拍完所有照片！"，这时候可以去预览区查看效果了。

★ STEP 4 — 预览、重拍与下载
右侧面板：STEP 3 PREVIEW 区域
1👁️
查看右侧预览区

每拍完一张，照片就会出现在右侧"PREVIEW"面板的对应格子里。你可以实时看到大头贴拼图的整体效果，包括边框颜色和贴纸叠加后的样子。

STEP 3 — PREVIEW
😁
😎
🤪
↑ 4格模式：已拍3格，第4格待拍（斜纹表示空格）
2🔄
点击某一格重拍

对预览区中的某张照片不满意？直接点击那一格，会弹出"确定要重拍这个照片吗？"的确认框。

确认后，只有这一格会被清空，其他格子的照片保留不变。不需要重新拍所有格子——这是 MemePhoto 最实用的功能之一。

🔄
重拍流程：点击预览格 → 确认弹窗 → 摄像头恢复待机 → 重新点"拍照"按钮 → 补拍这一格。
3💾
点击"下载组合照片"

全部格子拍满并满意后，点击中间面板底部的"下载组合照片"按钮。

网站会将所有格子的照片合并成一张带边框的大头贴拼图，自动下载到你的设备。文件名为 memephoto.png，保存在浏览器默认的下载文件夹中。

下载成功后会播放一声拍立得相机的快门声 🎵

✅
下载后的图片格式：PNG 格式，高清无损。照片边缘带有 @memephoto.com.cn 水印小字（位于边框内侧），不影响整体观感。

  常见问题排查 | Troubleshooting
遇到问题？先看这里
点了"启动摄像头"没有反应，或者摄像头显示黑屏 ▶
最常见原因：浏览器摄像头权限被阻止。
解决步骤：① 点击浏览器地址栏左侧的锁形图标（🔒）→ 找到"摄像头"→ 改为"允许" → 刷新页面。
② iPhone 用户：设置 App → Safari → 摄像头 → 选择"允许"。
③ 如果摄像头正在被其他应用使用（如视频会议软件），请先关闭那个应用再尝试。
④ 换用 Chrome 浏览器重试，Chrome 对摄像头 API 的支持最稳定。
拍照后预览区显示空白，或者下载的图片是空的 ▶
原因：摄像头画面还没有完全加载就点击了"拍照"。
解决方法：点"启动摄像头"后，等待摄像头画面完整显示在中间框架内（通常1–3秒），确认看到自己的脸后再点"拍照"。如果仍然空白，刷新页面重试，并确保摄像头权限已允许。
贴纸或边框图片一直加载中，转圈不显示 ▶
部分图片资源通过 CDN 加载，在某些网络环境下可能较慢。
解决方法：① 等待 5–10 秒，通常会完成加载。② 刷新页面重试。③ 如果你在使用企业/学校网络或代理，可能存在访问限制，建议切换到手机热点试试。如果问题持续，请通过联系页面告知我们，帮助改善加载方案。
手机上打开网站布局混乱，或者摄像头画面很小 ▶
MemePhoto 对桌面端优化更好。手机使用建议：
① 横屏使用——转为横屏后布局会更宽敞，摄像头区域更大。
② 使用 Chrome 而非系统默认浏览器。
③ 在微信中打开时，点右上角"…"→"在浏览器中打开"。
手机端体验优化是我们正在持续改进的工作，感谢你的耐心。
下载按钮点击后没有任何反应 ▶
最常见原因：还没有拍满所有格子。
检查右侧预览区，确认所有格子都有照片（没有斜纹空格）。如果仍无反应，检查浏览器是否阻止了文件下载（浏览器右上角可能有下载权限提示），点击允许即可。Safari 用户如果无法下载，可以尝试长按下载链接选择"存储图像"。
拍出来的照片方向反了（文字是镜像的） ▶
这是正常的设计——MemePhoto 使用镜像模式拍摄（和自拍一样），拍出来的照片是左右镜像的。这是因为大多数人觉得看镜像的自己更自然，更接近日常习惯。
如果你的贴纸上有文字，该文字在照片中会是反向的。建议使用无文字贴纸或透明贴纸来避免这个问题。
如何把照片添加到主屏幕，像 App 一样使用？ ▶
iPhone / iPad（Safari）：打开 memephoto.com.cn → 点击底部分享图标 → "添加到主屏幕" → 点"添加"。
Android（Chrome）：打开网站 → 点右上角"…"菜单 → "添加到主屏幕"。
添加后图标会出现在桌面，点击即可像 App 一样打开，无需重新在浏览器输入网址。
